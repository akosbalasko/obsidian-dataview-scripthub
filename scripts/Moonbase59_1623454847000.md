# `=this.ingredient`
&#x2026; description, notes &#x2026;

### Used in these recipes

```dataviewjs
dv.list(dv.pages()
    .where(p =&gt; p.recipe &amp;&amp; p.ingredients.map(e =&gt; e[0])
        .indexOf(dv.current().ingredient) &gt;= 0)
    .file.link
);
```
